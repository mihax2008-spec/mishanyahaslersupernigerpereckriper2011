# Рецерпты от AI-шефа
🍳 AI Chef Telegram Bot
AI Chef — это умный кулинарный помощник в Telegram, работающий на базе нейросети GPT-4o-mini. Бот выступает в роли профессионального шеф-повара с 25-летним стажем, генерируя уникальные рецепты по любым запросам пользователя.
🚀 Основные функции
Рецепты по запросу: Просто напиши «Ужин из курицы за 20 минут» или «Веганский десерт», и бот составит пошаговый план.
Случайный рецепт: Кнопка «🍽 Случайный рецепт» для тех, кто не знает, что приготовить сегодня.
Красивое оформление: Все рецепты приходят с четкой структурой: время приготовления, порции, калорийность, список ингредиентов и советы шефа.
HTML-форматирование: Текст легко читать благодаря выделенным заголовкам и эмодзи.
Интеграция через API Tunnel: Стабильный доступ к современным LLM моделям.
🛠 Технический стек
Язык: Python 3.x
Библиотека: python-telegram-bot (v20+)
AI Мозг: OpenAI API (GPT-4o-mini) через прокси aitunnel.ru
Управление окружением: python-dotenv
📋 Как запустить
Клонируйте репозиторий:
bash
git clone https://github.com
cd ai-chef-bot


Установите зависимости:
bash
pip install python-telegram-bot openai python-dotenv


Настройте переменные окружения:
Создайте файл .env в корне проекта и добавьте:
env
TELEGRAM_TOKEN=твой_токен_от_BotFather
OPENAI_API_KEY=твой_ключ_от_aitunnel


Запустите бота:
bash
python bot.py


🔄 Логика работы (Flowchart)
Бот работает по следующему циклу:
Прием данных: Сообщение от пользователя или нажатие кнопки.
Обработка промпта: Текст пользователя объединяется с CULINARY_SYSTEM_PROMPT.
Запрос к AI: Бот имитирует «печать» и отправляет данные в нейросеть.
Выдача: Пользователь получает красиво оформленный рецепт в формате HTML.
